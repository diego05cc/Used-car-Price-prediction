Used Car Price Prediction

Applying Linear Regression to estimate the market value of pre-owned vehicles based on measurable attributes.

Problem Description

The Challenge

Determining the fair price of a used car is a complex decision influenced by many interdependent variables. Buyers rely on subjective assessments, while sellers often lack objective benchmarks, leading to systematic mispricing. Traditional approaches — consulting price guides or relying on dealer experience — are slow, inconsistent, and difficult to scale.

Machine Learning enables a data-driven, reproducible pricing model: given a set of vehicle characteristics, the system returns a statistically grounded price estimate, reducing uncertainty for both buyers and sellers.

Type of ML

Supervised Learning — Regression. The model is trained on labelled historical records (vehicle attributes + known sale price) and learns to map inputs to a continuous numeric output.

Input Features

  • Registration year
  • Mileage (km)
  • Engine displacement (cc)
  • Brand / manufacturer
  • Fuel type (petrol, diesel, hybrid)
  • Transmission type
  • Number of previous owners
  • Vehicle condition rating

Expected Output

A single continuous numeric value representing the predicted market price of the vehicle in the local currency. The output is the model's best estimate given the supplied feature values.

Real-World Relevance

Supports fair negotiation between buyers and sellers, helps dealerships set competitive prices, and underpins online listing platforms that display automatic price appraisals.

Algorithm

Why Linear Regression?

Linear Regression assumes a linear relationship between the input features and the target variable (price). Despite its simplicity, it is highly interpretable — each coefficient directly quantifies how a one-unit change in a feature affects the predicted price — making it well-suited for an academic demonstration.

The model minimises the Residual Sum of Squares (RSS) during training, identifying the optimal weight for each feature. Performance is then evaluated using R², MAE, and RMSE on a held-out test set.
