Under the Hood

Linear Regression — simplified

Linear Regression finds the best straight line through a scatter of data points. For car pricing, the model learns how each feature (mileage, age, brand…) pushes the price up or down.

During training the algorithm adjusts its internal weights to minimise the error between predicted and actual prices. Once trained, a prediction is a single, fast arithmetic calculation.

Fit quality is measured by the R² score — a value of 1.0 means perfect predictions. Our model reaches 0.94.

Core formula

ŷ = β₀ + β₁x₁ + β₂x₂ + … + βₙxₙ
  • ŷ — predicted price
  • β₀ — intercept (base price)
  • βᵢ — learned weight per feature
  • xᵢ — input feature value
